# Markdown List Use Cases

## Unordered Lists

- First item
- Second item
- Third item
- Fourth item

* First item
* Second item
* Third item
* Fourth item

* First item
* Second item
* Third item
    - Indented item
    - Indented item
* Fourth item

## Ordered Lists

1. First item
2. Second item
3. Third item
4. Fourth item

1. First item
1. Second item
1. Third item
4. Fourth item

## Mixed Lists

1. First item
1. Second item
1. Third item
    - Indented item
    - Indented item
1. Fourth item
