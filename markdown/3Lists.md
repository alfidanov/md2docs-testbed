# Markdown List Use Cases

## Unordered Lists

Unordered list with -

- First item 11111
- Second item 22222
- Third item 33333
- Fourth item 44444

Unordered list with *

* First item
* Second item
* Third item
* Fourth item

Unordered list with * and -

* First item
* Second item
* Third item
    - Indented item
    - Indented item
* Fourth item

## Ordered Lists

Ordered list with 1. 2. 3. 4.

1. First item
2. Second item
3. Third item
4. Fourth item

Ordered list with 1. 1. 1. 1.

1. First item
1. Second item
1. Third item
1. Fourth item

## Mixed Lists

1. First item
1. Second item
1. Third item
    - Indented item
    - Indented item
1. Fourth item
